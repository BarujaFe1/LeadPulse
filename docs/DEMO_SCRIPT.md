# Demo script (3–5 minutes)

**URL:** https://leadpulse-two.vercel.app  
**Audience:** recruiter / hiring manager (analytics engineering / data product / full-stack analítico)

1. **Hook (20s)** — “Negócios locais perdem leads no WhatsApp depois de já terem pago CAC. O LeadPulse é um cockpit de follow-up, não um CRM.”
2. **KPIs (40s)** — apontar leads esquecidos, mediana de 1ª resposta e receita em risco. Enfatizar: snapshot sintético de 12 leads.
3. **Radar (45s)** — abrir tarefas `critical` (sem 1ª resposta / after-hours).
4. **Funil + perda (40s)** — funil simples e motivos de perda (processo, não culpar o lead).
5. **Heurística (60s)** — priorizar a mensagem quente; ler score, rationale e painel “Como o opportunity score funciona”. Deixar explícito: **não é IA/LLM**.
6. **Close (30s)** — escopo: CSV/form primeiro; WhatsApp oficial é Fase 2; zero scraping.

## Claims permitidos

- Lab demo WhatsApp-first com dados sintéticos
- KPIs de resposta / forgotten leads / score heurístico
- Frontend-first na Vercel + FastAPI opcional

## Claims proibidos

- “IA que classifica leads”
- “Integração oficial WhatsApp em produção”
- “CRM enterprise / premium”
- Métricas de clientes reais
